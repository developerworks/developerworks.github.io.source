## 入门

```
# 创建新型项目
mix new your_project
cd your_project
# 添加依赖 '{:sugar, github: "sugar-framework/sugar"}'
vi mix.exs
# 获取依赖并编译
mix do deps.get, deps.compile
# 添加Sugar文件
mix sugar.init
# 启动服务器
mix server
```
